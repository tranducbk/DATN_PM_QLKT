# 🎖️ QLKT - Hệ thống Quản lý Khen thưởng

## 📋 Giới thiệu

**Phần mềm Quản lý khen thưởng** là hệ thống quản lý khen thưởng toàn diện cho Học viện Khoa học Quân sự, giúp quản lý danh hiệu, thành tích khoa học và tính toán khen thưởng tự động.

This project is proprietary software for Vietnam Military Science Academy.
